## Run Locally

Clone the project

```bash
  git clone https://github.com/atinder11/Iron_Edge_Gym.git
```

Install dependencies in the respective directories

```bash
  npm install
```

```bash
  yarn
```


Run the application in  on your localhost

```bash
  yarn run dev
```
